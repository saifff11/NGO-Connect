NGO Connect is a web platform that connects NGOs, Volunteers, and Donors.

## Tech Stack
- Frontend: React.js
- Backend: Django REST Framework
- Database: SQLite
- Authentication: JWT / Session Auth

## Features
- Multi-role system (NGO, Volunteer, Donor)
- NGO project management
- Volunteer registration
- Donation system
- Admin dashboard

## Setup Instructions

### Backend
cd backend
pip install -r requirements.txt
python manage.py runserver

### Frontend
cd frontend
npm install
npm start
